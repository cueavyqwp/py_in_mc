# python in minecraft

## 原理

使用`subprocess`模块开启一个MC服务器子进程

使用`threading`和`prompt`实现命令行操作

使用`/data`来获取数据

使用`/say`来发送信息

## 条件

你需要以下条件

- `java` 17+
- `python` 3.6+
- `Minecraft` 1.16+

## 运行
请确保你已配置好`Java`与`python`

并且在同目录下有一个名叫`server.jar`的1.16+服务端 且同意了`EULA`

运行`main.py`

`python main.py`或`python3 main.py`

使用`书与笔`写好代码后不要署名 直接放入讲台

输入`!run {讲台坐标}`

即可看到效果

## 关闭服务器

使用`/stop`关闭服务端

使用`q`退出程序
